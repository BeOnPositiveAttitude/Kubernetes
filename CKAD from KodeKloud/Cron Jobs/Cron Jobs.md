CronJob позволяет нам запланировать периодический запуск какой-либо задачи
```
* * * * * выполняемая команда
- - - - -
| | | | |
| | | | ----- день недели (0—7) (воскресенье = 0 или 7)
| | | ------- месяц (1—12)
| | --------- число (1—31)
| ----------- час (0—23)
------------- минута (0—59)
```

Смотреть список CronJobs: `kubectl get cronjobs`
